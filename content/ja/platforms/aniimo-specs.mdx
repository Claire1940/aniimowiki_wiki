### 主要属性の解説

アニモのステータスは簡素化されていますが、スキルツリーと組み合わせることで深い戦略性が生まれます。物理攻撃力と魔法攻撃力を分ける従来のRPGとは異なり、アニモの「攻撃」ステータスはすべてのスキルタイプのダメージ出力を制御するため、装備の選択が簡略化される一方で、ロールの専門化が強調されています。

| ステータス | 機能 | 戦闘への影響 |
| :--- | :--- | :--- |
| **HP** | 体力 | 力尽きるまでの総生存能力を決定します |
| **攻撃** | 攻撃力 | 物理スキルと魔法スキルの両方のダメージを増加させます |
| **物理防御** | 物理防御 | 近接攻撃や遠距離攻撃から受けるダメージを軽減します |
| **魔法防御** | 魔法防御 | 属性魔法やエーテル魔法によるダメージを緩和します |
| **回復** | 回復率 | HPとEP（エレメンタルパワー）の回復速度を制御します |

### 遺伝的ポテンシャル・ティア

捕獲したアニモを報告すると、システムはその遺伝的構成を評価します。この分類は永続的で変更できないため、「パーフェクト」の個体は対人戦において非常に重宝されます。

| 素質ティア | 成長ポテンシャル | 推奨事項 |
| :--- | :--- | :--- |
| **コモン** | 標準 | 序盤の進行や合成素材として使用 |
| **良質** | 中程度 | 中盤のクエストや探索において信頼できる |
| **エリート** | 高い | アルファボス戦や高難易度PvEに適している |
| **パーフェクト** | 最大 | エンドゲームやPvPビルドのゴールドスタンダード |

<Callout type="tip" title="プロのヒント">
新しいアニモを捕獲した後は、必ず「能力」セクションを確認してください。DPS重視のクリーチャーの場合、序盤は「剛力（Might）」にスキルポイントを投資するのが、攻撃ステータスを最大化する最も効率的な方法です。
</Callout>

<Section icon={Zap} title="属性相性とタイプマッチアップ">

アニモの世界は9つの異なる属性タイプで構成されています。属性相性に関する**アニモのスペック**をマスターできるかどうかが、20分かかるボス戦を5分で終わらせられるかどうかの分かれ目になります。属性優位はダメージを2倍にし、これは「ブレイク」状態などの他の補正と重複します。

### 9つの属性

各アニモは単一タイプまたはデュアルタイプになります。デュアルタイプのクリーチャーは汎用性が高い反面、弱点も多くなる可能性があります。

| 属性 | 主な強み | 一般的なカウンター |
| :--- | :--- | :--- |
| **火** | 高い瞬間火力 | 水、土 |
| **水** | 持続力/回復 | 草、雷 |
| **草** | サポート/デバフ | 火、氷 |
| **土** | 高い物理防御 | 水、風 |
| **風** | 機動力/空間制御 | 氷、雷 |
| **氷** | クラウドコントロール（凍結） | 火、土 |
| **雷** | 速度/チェイン攻撃 | 土、風 |
| **光** | 回復/バフ | 闇 |
| **闇** | 高い魔法防御/EPドレイン | 光 |

<Callout type="warning" title="バトルインジケーター">
戦闘中のテキストポップアップに注意してください。「Excellent」は属性優位（ダメージ2倍）を示し、「Bad」は劣勢（ダメージ半分）を示します。
</Callout>

### 空間相関：じゃんけんシステム

属性だけでなく、アニモには重み付けされた「じゃんけん」のように機能する「空間相関」システムが採用されています。これは、クリーチャーの物理的な位置関係に基づいて攻撃がどのように相互作用するかを決定します。

<CardGrid cols={3}>
  <Card title="飛行 vs 地上">
    飛行中のターゲットに当てるには遠距離攻撃が不可欠です。地上の近接攻撃は、空中の敵に対してミスしたりダメージが減少したりすることがよくあります。
  </Card>
  <Card title="潜行 vs 震動">
    潜行中のアニモはほとんどの標準的な攻撃が無効です。「震動（Tremor）」スキルを持つクリーチャーを使用して、地上に引きずり出す必要があります。
  </Card>
  <Card title="近接 vs 遠距離">
    近接スキルは高い「ブレイク」ポテンシャルを持ちますが、接近するリスクを伴います。一方、遠距離スキルは安全性を提供しますが、衝撃力は低くなります。
  </Card>
</CardGrid>

<Section icon={Swords} title="戦闘ロールとチームシナジー">

アルファアニモに立ち向かうには、バランスの取れたパーティが必要です。DPSユニットを4体並べたくなるかもしれませんが、ゲームのメカニクスは、エレメンタルパワー（EP）とブレイクメーターを効果的に管理できる多様な編成に報酬を与えます。

### 5つの戦闘ロール

| ロール | 主な目的 | 重点ステータス |
| :--- | :--- | :--- |
| **DPS** | 純粋なダメージ出力 | 攻撃、回復 |
| **ブレイク** | ブレイクメーターを削る | 攻撃、物理防御 |
| **サポート** | 味方のバフ/敵のデバフ | 魔法防御、回復 |
| **リジェネ** | エレメンタルパワー（EP）の回復 | 回復、HP |
| **ヒーラー** | チームのHP回復 | HP、魔法防御 |

<Callout type="success" title="メタ戦略">
「リジェネ」ロールは見落とされがちですが、非常に重要です。**ウィチェン（Wichen）**のようなクリーチャーは、ヒットするたびにチームのEPを回復するデバフを付与できるため、DPSが強力なスキルを際限なく連発できるようになります。
</Callout>

### ボス戦の理想的なチーム編成

高レベルのPvEコンテンツでは、以下の4スロットのパーティ構成をお勧めします。

<Steps>
  <Step num="1" title="ブレイカー">
    戦闘はブレイカーから始めましょう。目的は敵の黄色いバーをできるだけ早く削りきることです。2倍のブレイク効率を得るために、属性優位のあるブレイカーを使用してください。
  </Step>
  <Step num="2" title="サポート/リジェネ">
    リジェネユニットに入れ替えて、EP回復バフを付与します。これにより、弱い通常攻撃に頼ることなく、チームがスキルを放つための十分なエネルギーを確保できます。
  </Step>
  <Step num="3" title="DPSペイロード">
    「ブレイク」状態になったら、メインDPSに切り替えます。この時がアルティメット攻撃を使う絶好の機会です。属性（2倍）とブレイク（2倍）により、4倍のダメージを与えることができます。
  </Step>
  <Step num="4" title="ヒーラー（緊急用）">
    ミスをカバーするために、最後のスロットにヒーラーを配置しておきます。グラスウィングのような一部のヒーラーは属性耐性も提供し、ボスのアルティメット攻撃中にチームを生存させます。
  </Step>
</Steps>
