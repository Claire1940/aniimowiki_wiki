<Section icon={Shield} title="戦闘と育成のメカニクス">

**アニモ デモ**での育成は、単なるレベルアップにとどまりません。このゲームは「コンパニオン」とプレイヤーの間の絆を強調しています。「コンパニオン・プロミス」システムを通じて、プレイヤーは自身の進歩を示すユニークなアバターフレームやコスメティックアイテムを解放できます。戦闘はテンポが速くなるよう設計されており、プレイヤーは回避、パリィを行い、アニモの特殊能力をタイミングよく発動させて最大の効果を得る必要があります。

<Callout type="info" title="フォームの切り替え">
戦闘中にフォームを切り替えることで、特定のクールダウンをリセットしたり、一時的な属性耐性を得たりすることができます。
</Callout>

以下の表は、最初の数時間のプレイで遭遇する異なるフォームの有用性を比較したものです。

| フォームタイプ | 最適な用途 | 主な利点 |
| :--- | :--- | :--- |
| **ノーマルフォーム** | 一般的な探索 | バランスの取れたステータス、低エネルギーコスト |
| **サンダーストーム** | 複数ターゲットとの戦闘 | 高い範囲ダメージ、スタン能力 |
| **プリズマナ** | ボス戦 | 優れた属性貫通力 |
| **レインストーム** | サポート/防御 | 回復能力、ステータス浄化 |
| **マッドフラット** | 地形のナビゲート | 水中での移動速度向上 |

プレイヤーは、アイドールのエコシステムで提示される多様な課題に対処するために、早い段階で少なくとも2つのアニモタイプをマスターすることに集中すべきです。デモ版では、特に異なるチーム構成を試すプレイヤーに報酬が与えられます。

</Section>
