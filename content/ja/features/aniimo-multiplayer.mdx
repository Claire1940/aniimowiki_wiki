<Section icon={Swords} title="対戦プレイ：ダイヤモンド・エッグ強奪戦とPvP">

**アニモ マルチプレイヤー**における対戦の頂点が**ダイヤモンド・エッグ強奪戦**です。危険なロスト・アイルズを舞台に、3人編成の6チームがダイヤモンド・エッグを手に入れるために競い合います。これは単なるキャプチャー・ザ・フラッグではなく、移動アビリティとチーム編成への深い理解が求められます。

このモードでは、プレイヤーはトワインアビリティ（滑空、登攀、水泳）を駆使して地形を進み、中央の目標地点を目指します。チームが卵を確保すると、その運搬者は格好の標的となります。残りの2人のチームメイトは、ニンビ（Nimbi）のような防御型アニモで運搬者を守るか、エンバーパップ（Emberpup）のような攻撃型で追跡者を迎撃し、保護を提供しなければなりません。

| 特徴 | ダイヤモンド・エッグ強奪戦 | ホロバトル・インターリンク |
| :--- | :--- | :--- |
| **プレイヤー数** | 18名 (3人×6チーム) | 1～3名 |
| **主な目標** | ダイヤモンド・エッグの確保と護衛 | 敵やボスのウェーブを撃破 |
| **マップの場所** | ロスト・アイルズ | ポラリス研究所 |
| **主な報酬** | ダークラー形態のアニモ、レアな卵 | 強化素材、レゾネーター |
| **戦闘スタイル** | PvEvP (クリーチャー + プレイヤー) | PvE (シミュレーション) |

<Callout type="warning" title="戦略アラート">
ダイヤモンド・エッグ強奪戦において、チームを離れて突出することは敗北への近道です。トリオのうち少なくとも1人は、スカウトや緊急時の運搬役として機動力重視のアニモを編成するようにしてください。
</Callout>

強奪戦以外にも、**ホロバトル・インターリンク**では、より構造化された対戦環境が提供されます。主にシミュレーションのクリアを目的としていますが、リーダーボードやスコアアタックの要素により、コミュニティに対して自分の戦闘効率を証明する場となります。

</Section>

<Section icon={Zap} title="協力型チャレンジ：アルファボスとサンクタム">

競争よりも協調を好むプレイヤーのために、**アニモ マルチプレイヤー**は広範なPvEコンテンツを用意しています。イディル大陸には、通常の個体よりも大幅に高いHPと壊滅的な攻撃パターンを持つ巨大な**アルファ・アニモ**が生息しています。水属性のババスク（Bubbask）や強大なマインスパイン（Minespine）などの遭遇戦では、ヘイト（敵視）を管理し、隙を作るために十分なブレイクダメージを与えるフルパーティのパスファインダーが必要になることがよくあります。

**サンクタムパズル**は、また別の形の協力を象徴しています。これらは2人協力プレイ用に設計された専用ダンジョン（例：*サンクタム：仕掛けの迷宮*）です。プレイヤーは、火属性で蒸気機関を動かし、その間に風属性が浮遊プラットフォームを操作するなど、アニモ独自の環境アビリティを駆使してゴールを目指します。

<Steps>
  <Step num="1" title="パーティを結成する">
    ソーシャルメニュー（Esc > ソーシャル）を使用して友人を招待するか、アストラにあるマッチングキオスクを使用してパートナーを探します。
  </Step>
  <Step num="2" title="補完的な編成を選択する">
    チームで複数の属性をカバーできるようにします。パートナーが高DPSの火属性を出す場合は、サポート用の風属性や、ブレイクを狙うための岩属性を検討してください。
  </Step>
  <Step num="3" title="エンカウントを開始する">
    マップ上のアルファボスのアイコンへ移動するか、サンクタムのポータルに入ります。開始するにはパーティメンバー全員が揃っている必要があります。
  </Step>
  <Step num="4" title="トワインの使用を連携させる">
    トワインモードに切り替えるタイミングを伝え合いましょう。ボスの「アルティメット」攻撃を回避したり、コンボを実行したりするには、直接操作が不可欠です。
  </Step>
</Steps>

| ボス名 | 属性 | 推奨ロール | 主なドロップ |
| :--- | :--- | :--- | :--- |
| **ババスク** | 水 | 電気 / 草 DPS | 純粋なルミナイト |
| **マインスパイン** | 岩 / 闇 | 水 / 聖 ブレイク | 進化の結晶 |
| **コラリス** | 水 / 氷 | 火 / 電気 DPS | レアなプリズマナの卵 |
| **スパークリング・アルファ** | 変動 | 属性カウンター | きらめく宝石 |

<Callout type="tip" title="プロのヒント">
協力プレイでアルファボスと戦う際は、1人を「ブレーカー」役に任命しましょう。そのプレイヤーの唯一の仕事は、高いブレイク能力を持つアビリティでボスを怯ませることであり、その隙に他のプレイヤーが最も強力なトワインコンボを叩き込めるようにします。
</Callout>

</Section>

<Section icon={Map} title="ソーシャルハブとホームランドシステム">

アニモにおけるマルチプレイヤーは戦闘だけではありません。都市**アストラ**は主要なソーシャルハブとして機能し、何百人ものパスファインダーが集まり、レアなアニモを披露したり、資源を取引したりできます。ここは装飾店、アクセサリーショップ、ポラリス研究所などが立ち並ぶコミュニティの中心地です。

**ホームランド**システムは、ソーシャル体験に個人的な彩りを添えます。すべてのプレイヤーは、カスタマイズや拡張が可能なRV（キャンピングカー）連結型の拠点を所有しています。友人をホームランドに招待して、アニモのコレクションを見せたり、作物の栽培を手伝ってもらったり、カスタムデザインされた空間でくつろいだりすることができます。

<CardGrid cols={2}>
  <Card title="アニモ交換">
    - **トレード**: 捕まえたアニモを他のプレイヤーと交換してインデックスを完成させましょう。
    - **ギフト**: 友人の進行を助けるために、資源や一般的なアイテムを送りましょう。
    - **マーケットプレイス**: アストラのソーシャル地区に出品されているクリーチャーを閲覧できます。
  </Card>
  <Card title="ホームランド訪問">
    - **展示**: 進化したガンマ形態やノヴァ形態を庭に飾りましょう。
    - **協力農業**: 友人が水やりを手伝ったり、あなたのアニモと触れ合ったりできます。
    - **カスタマイズ**: あなたのインテリアデザインのレイアウトをコミュニティと共有しましょう。
  </Card>
</CardGrid>
