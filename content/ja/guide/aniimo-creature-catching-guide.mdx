「分布表示」機能は、熱心なハンターにとって最も強力なツールです。この機能をクリックすると、マップ上に小さな黄色のスポットが表示されます。これらのスポットは、選択したアニマが出現しやすい正確な座標を示しています。目的もなく歩き回る代わりに、最も近い「キューブ」やウェイポイントにファストトラベルし、これらの高密度ゾーンに直接向かうことができます。

| マップ機能 | 主な機能 | 戦術的メリット |
| :--- | :--- | :--- |
| **近接図** | 現在のゾーンにいるすべてのアニマをリスト表示 | 違う地域で時間を無駄にするのを防ぐ |
| **トリガー条件** | 必要な時間や天候を表示 | 夜間限定や嵐限定の出現に不可欠 |
| **分布表示** | 特定の出現座標をマーク | ターゲットへの直接的なルート構築が可能 |
| **ゾーン境界** | 近接リストの対象エリアを定義 | 出現が切り替わる境界を正確に特定できる |

<Section icon={Clock} title="環境トリガーと時間の操作">

アニモの出現は固定されていることは滅多にありません。ほとんどの生物は、世界に出現する前に満たさなければならない環境トリガーと結びついています。最も一般的なトリガーは時間帯です。例えば、通常形態のイグニス（Ignis）は夜間にしか出現しません。昼間に出現場所に到着しても、そのエリアは空っぽに見えるか、他の生物の「幼体」バージョンしか存在しません。

幸いなことに、このゲームでは手動で時間を操作することができます。メニュー内の「昼夜切り替え」トグルにアクセスすることで、時間を進めて必要な時間帯にジャンプできます。これはコレクションを効率的に完成させるために不可欠です。ただし、時間を変更しても現在の戦闘状態が常にリセットされるわけではなく、出現ノードの近くに立ちすぎていると、移行中に一部のアニマが襲いかかってくる（「走ってくる」）可能性があることに注意してください。

| トリガータイプ | 具体例 | 捕獲戦略 |
| :--- | :--- | :--- |
| **時間依存** | 夜間のみ、夜明けのみ | タイムジャンプツールを使用して待ち時間をスキップ |
| **天候依存** | 雨、霧、晴れ | ミニマップの横にある天候アイコンを確認 |
| **チェイン依存** | ハイランド形態を15匹捕獲 | 長時間の捕獲セッションに備えてバッグを整理 |
| **アクション依存** | 生徒をX人倒す | メインの出現が現れる前に戦闘に集中する |

<Callout type="warning" title="天候の罠">
「偽の夜」に注意してください。昼間に雨が降っていると空が暗くなり、夜のように見えることがありますが、システム上はまだ太陽が雲の後ろに出ています。夜行性のアニマを探す前に、必ずメニューで正確なゲーム内時間を確認してください。
</Callout>

</Section>

<Section icon={Search} title="姿を消すアニマの特定と捕獲">

この**アニモ捕獲ガイド**に登場するすべての生物が、分かりやすい場所に立っているわけではありません。高度なアニマ、特に「影に隠れている」や「獲物をおもちゃにする」と説明されているアニマは、クローキング（ステルス）機能を使用しています。分布ポイントに到達しても何も見当たらない場合、出現バグだと決めつけないでください。代わりに、環境の手がかりを探しましょう。

姿を消しているアニマは、かすかに揺らめくオーラを残したり、茂みから光る目で見つめてきたりすることがよくあります。また、同じ種のより小さな、ステルス化していないバージョンと「訓練」しているところを発見されることもあります。これらの生物と交戦するには、多くの場合「背後からの不意打ち」アプローチや、逃げられる前にアンティポード（主要な捕獲ツール）を素早く投げつける必要があります。

<Section icon={Target} title="ステップ・バイ・ステップ捕獲プロセス">

効率を最大化するために、野生で新しい生物に遭遇した際は、以下の標準化されたワークフローに従ってください。

<Steps>
  <Step num="1" title="マップ図を確認する">
    目的の地域のマップを開き、捕獲したいアニマを見つけます。「トリガー条件」を注意深く読み、間違った時間帯に探していないか確認します。
  </Step>
  <Step num="2" title="環境を調整する">
    条件が「夜間」で、現在が昼の場合は、タイムジャンプ機能を使用します。ジャンプを確定し、世界が正しい状態にリロードされるのを待ちます。
  </Step>
  <Step num="3" title="分布を特定する">
    「分布表示」をクリックし、黄色い点を探します。ナビゲーションの目印として、点が最も密集している場所に手動でウェイポイントを設定します。
  </Step>
  <Step num="4" title="捕獲を実行する">
    慎重にエリアに近づきます。アニマがステルス状態の場合は、クローキングのオーラを探します。アンティポードを使用して捕獲を成功させましょう。レア変異体の場合は、事前に「チェイン」要件を完了していることを確認してください。
  </Step>
</Steps>

| 捕獲段階 | 推奨アイテム | 成功率 |
| :--- | :--- | :--- |
| **初期接触** | スタンダードアンティポード | 45% - 60% |
| **弱らせたターゲット** | グレートアンティポード | 75% - 85% |
| **レア/プリズメイン** | ウルトラアンティポード | 90% + |
| **グループ捕獲** | スプラッシュアンティポード | 変動あり |

<Callout type="info" title="ファストトラベルの裏技">
ベータ版の一部のエリアでは、戦闘中であってもファストトラベルが可能です。戦闘が不利になったり、誤って間違ったターゲットと交戦してしまったりした場合は、マップを使って近くのキューブに直接ワープし、遭遇戦をリセットしましょう。
</Callout>

</Section>

<Section icon={Trophy} title="レア変異体と連続捕獲（チェイン）の目標">

**アニモ捕獲ガイド**における究極の挑戦は、「プリズメイン（Prismane）」形態を獲得することです。これらは単に夜になるのを待つだけでは出現しない高ランクの変異体です。彼らを世界に「強制的に」出現させるには、プレイヤーの特定のアクションが必要になります。最も一般的な要件は「連続捕獲（チェイン）」です。

例えば、プリズメイン・イグニタス（Prismane Ignitus）は、他の生物を捕獲したりゾーンを離れたりすることなく、ハイランド形態を連続で15匹捕獲した後にのみ出現します。これには忍耐と大量のアンティポードの備えが必要です。ゲームのマップUIにはこれらの条件が非常に明確に記載されているため、ハントを開始する前に必ず詳細な説明を読んでください。

| **アルファ形態** | ベリーハード | ワールドボスのトリガー |

<Callout type="warning" title="チェイン切れに注意">
異なる種のアニマを捕獲すると、通常は現在のチェインが途切れてしまいます。プリズメインを狙う場合は、たとえ他に入手困難なレア生物が現れたとしても、目標数に達するまでは完全に無視してください。
</Callout>
