<Section icon={Shield} title="システム要件と互換性">

アニモの「荒野の飛行」や「無重力」戦闘をパフォーマンスの低下なしに体験するには、ハードウェアが特定のベンチマークを満たしている必要があります。本作は、公式資料で言及されている「移ろう光」を描写するために高度なライティングシェーダーを使用しており、古いGPUでは負荷がかかる場合があります。

| コンポーネント | 最小要件 | 推奨スペック |
| :--- | :--- | :--- |
| **OS** | Windows 10 64-bit | Windows 11 64-bit |
| **プロセッサ** | Intel i5-8400 / AMD Ryzen 5 2600 | Intel i7-10700 / AMD Ryzen 7 3700X |
| **メモリ** | 8 GB RAM | 16 GB RAM |
| **グラフィックス** | NVIDIA GTX 1060 / RX 580 | NVIDIA RTX 3060 / RX 6700 XT |
| **DirectX** | Version 11 | Version 12 |
| **ストレージ** | 50 GB SSD空き容量 | 50 GB NVMe SSD空き容量 |

「並行空間」への移行をロード画面なしでスムーズに行うために、**アニモ Epic Games**をSSDで実行することを強く推奨します。古いHDDを使用しているプレイヤーは、「魂とあなた」の同期シーケンス中にスタッタリング（カクつき）を経験する可能性があります。

<Callout type="warning" title="パフォーマンスに関する注意">
高強度の「無重力」戦闘フェーズ中に視覚的な不具合を避けるため、グラフィックスドライバーが最新の2026年バージョンに更新されていることを確認してください。
</Callout>

<Section icon={Target} title="アニモのインストールとセットアップ方法">

**アニモ Epic Games**のローンチに向けたアカウント設定は簡単なプロセスですが、以前のクローズドベータセッションに参加したことがある場合は、アカウントの連携に注意が必要です。

<Steps>
  <Step num="1" title="Epic Games Storeへのアクセス">
    Epic Gamesランチャーを開き、「アニモ（Aniimo）」を検索します。ゲームをライブラリに追加してください。2026年のプリロード期間中であれば、すぐにコアアセットのダウンロードを開始できます。
  </Step>
  <Step num="2" title="アカウント同期">
    初回起動時に、「アニモ・ソウルID（Aniimo Soul ID）」の作成を求められます。このIDをEpic Gamesアカウントにリンクして、「恐れ知らずの心（Fearless Heart）」の称号などの事前登録報酬が正しく適用されるようにしてください。
  </Step>
  <Step num="3" title="リージョン選択">
    物理的な場所に最も近いサーバーリージョンを選択してください。ソウルチェーンの「静かな呼び声（silent call）」は、リズム戦闘セクションでの最適なタイミングのために低レイテンシを必要とします。
  </Step>
  <Step num="4" title="グラフィックス調整">
    ゲーム内のベンチマークツールを実行してください。これにより、ハードウェアの能力に基づいて「揺らめく光」や「陽光の波」のエフェクトが調整され、安定した60 FPSが維持されます。
  </Step>
</Steps>

アニモにおける戦闘は、単なる力押しではありません。それはあなたの魂の「異なる鼓動」と「小さな輝き」に関するものです。ソウルシフトのタイミングをマスターすることで、一時的に並行空間に入って攻撃を回避し、重要な数秒間無敵になることができます。

| 戦闘スタイル | 主要属性 | 最適な用途 |
| :--- | :--- | :--- |

| **ソウルリーヴァー** | 俊敏 / 影 | 単体への高いバーストダメージ |
| **エコーウィーバー** | サポート / 音 | 味方の強化とリズム制御 |
| **ヴォイドウォーカー** | ユーティリティ / 重力 | クラウドコントロールと戦場操作 |
