捕獲は単にポッドを投げるだけではありません。希少な個体を見つけるには、生態系を研究する必要があります。夜行性のアニモは暗くなってからしか現れませんし、雨などの特定の天候は炎属性のアビリティを弱めるため、扱いやすくなります。

| 捕獲要素 | 成功への影響 | 推奨される行動 |
| :--- | :--- | :--- |
| **ステルス** | 大幅に上昇 | 背後から近づく、またはアニモが眠っている間に近づく |
| **ブレイクダメージ** | 中程度上昇 | 高ブレイクスキルを使用して、捕獲前にターゲットを気絶させる |
| **ポッドの品質** | 直接比例 | アルファ個体用に、常に上級または特級アニポッドを持ち歩く |
| **研究者レベル** | パッシブボーナス | アニモ図鑑（Aniilog）のタスクを完了して、捕獲スキルツリーを強化する |
| **状態異常** | 大幅に上昇 | 凍結や麻痺を付与して、ターゲットの逃亡を防ぐ |

<Callout type="tip" title="プロの極意">

<Section icon={Sparkles} title="進化ルートとレベルアップシステム">

**アニモハンドブック**における育成は、直接的なレベルアップと分岐進化の2つの主要な流れに分かれています。アニモの潜在能力を最大限に引き出すには、進化（変身）を開始する前に、現在のレベル上限（現在のベータフェーズでは通常レベル55）までレベルを上げる必要があります。

<CardGrid cols={3}>
  <Card title="経験値ジェム">
    - **瞬時にレベルアップ**
    - 新規捕獲した個体に最適
    - 宝箱やクエストで入手可能
  </Card>
  <Card title="ツイン戦闘">
    - **能動的なマスタリー**
    - 敵から経験値を獲得
    - 絆レベルを上昇
  </Card>
  <Card title="エリート訓練">
    - **高い効果**
    - 特定のステータス成長
    - プライマリーエネルギーが必要
  </Card>
</CardGrid>

進化は自動的には行われません。各クリーチャーには満たすべき特定の条件があります。例えば、ウィスピット（Wispit）をイグニタス（Ignitus）に進化させるには、レベル9に到達し、同じ形態のウィスピットを連続して3匹倒す必要があります。フルメンティス（Fulmentis）のような他の進化では、現在シーズンバトルパスから入手できる「フルミナスソウル（Fulminus Soul）」のような希少なアイテムが必要になる場合があります。

| 進化タイプ | 条件の例 | 必要なアイテム |
| :--- | :--- | :--- |
| **チャレンジ型** | 特定の敵を3匹倒す | なし |
| **アイテム型** | レベル上限に到達 | 属性ソウル/クリスタル |
| **ソーシャル型** | 絆レベルを最大にする | 特定の食べ物/ギフト |
| **環境型** | 特定のバイオームでレベルアップ | バイオーム固有の石 |

<Callout type="warning" title="進化に関する警告">
アニモを進化させると、進化前の形態の一部のスキルが使用できなくなる場合があります。進化を進める前に、必要なコアスキルをマスターしていることを確認してください。
</Callout>

</Section>

<Section icon={Trophy} title="完璧なアニモの育成：潜在能力と性格">

「完璧な」アニモは、その潜在能力スコアと性格特性によって定義されます。**アニモハンドブック**では、基礎ステータスが1から10のスケールで評価されます。完璧な個体は、推奨される2つのステータスで9または10のスコアを持ちます。フルメンティス（Fulmentis）のような攻撃特化型のアニモの場合、**攻撃（Attack）**と**ブレイク（Break）**、または**リジェネ（Regen）**に焦点を当てる必要があります。

<Steps>
  <Step num="1" title="潜在能力でソート">
    アニモ管理メニューを開き、「潜在能力スコア」でソートして、主要ステータスが8以上の個体を特定します。
  </Step>
  <Step num="2" title="推奨ステータスの確認">
    アニモのステータス画面の右側パネルを確認し、その種族に対してゲームが推奨している2つの属性（ステータス）を確認します。
  </Step>
  <Step num="3" title="サイケフルーツの使用">
    サイケフルーツを使用して性格を再抽選（リロール）します。戦闘重視のビルドでは、**会心率+4%**を付与する性格「**The N**」を狙いましょう。
  </Step>
  <Step num="4" title="潜在能力の覚醒">
    スターダストとクリスタルを使用して、潜在能力スコアを初期値（例：9）から覚醒最大値（例：19）まで引き上げます。
  </Step>
</Steps>

| 性格特性 | ステータスボーナス | 最適なビルド |
| :--- | :--- | :--- |
| **The N** | +4% 会心率 | 高DPS / 攻撃型ビルド |
| **アグレッシブ（Aggressive）** | +2% 攻撃 | 一般的な戦闘 |
| **頑丈（Sturdy）** | +5% 防御 | タンク / 前衛アニモ |
| **タクティカル（Tactical）** | +5% ブレイク | シールド破壊スペシャリスト |
| **エネルギッシュ（Energetic）** | +3% リジェネ | サポート / ヒーラービルド |

<Callout type="success" title="最適化成功">
潜在能力スコア10と正しい性格を組み合わせることで、通常捕獲した個体と比較して、アニモの戦闘力を40%以上向上させることができます。
</Callout>

</Section>

<Section icon={Swords} title="戦闘戦略とツインモード変身">

イディルでの戦闘はダイナミックで、**コマンドモード（Command Mode）**と**ツインモード（Twine Mode）**という2つの異なるスタイルが用意されています。この**アニモハンドブック**では、アルファ戦や高レベルのパスファインダーを突破するために、これら2つのモードの相乗効果（シナジー）を重視しています。

コマンドモードでは、プレイヤーは戦略家としてアニモに特定のスキルを使用するよう指示し、クールダウンを管理します。ツインモードでは、アニモと物理的に融合し、その動きを直接操作してリアルタイムのコンボを繰り出します。これは、ボスの強力な攻撃を回避したり、精密な攻撃を行ったりする際に不可欠です。

**属性相性のクイックリファレンス：**
- **炎**は**草**に有利
- **水**は**炎**に有利
- **草**は**水**に有利
- **雷**は**水**に有利

| 戦闘モード | 最適な用途 | 主な利点 |
| :--- | :--- | :--- |
| **コマンド** | 集団戦 | 戦術的なスキルの連携 |
| **ツイン** | ボス戦 | リアルタイムの回避とコンボ |
| **探索** | 世界の移動 | 特殊な移動（滑空/潜水など） |

<Callout type="info" title="属性相性のヒント">
[Esc] → [アニモ管理] から属性アイコンをクリックすることで、**属性相性パネル**を確認できます。様々な属性をバランスよく編成することが、アルファ戦を制する鍵となります。
</Callout>

</Section>

<Section icon={Shield} title="高度な強化：携帯アイテムとルーン">

**アニモハンドブック**における戦力育成の最終段階は、**携帯アイテム**と**ルーン**です。これらのアイテムは、大幅なステータスの固定値上昇やパーセンテージ上昇をもたらし、戦闘力を1,700台から2,400以上にまで引き上げることができます。

携帯アイテム（特にゴールドティアのもの）は最大レベル15まで強化できます。これらのアイテムのレベルを上げると、ルーンスロットが解放されます。ルーン自体には、固定ダメージ（True Damage）、会心ダメージ、HPなど、様々なステータスが付与されています。

| ルーンのタイプ | メインステータス | サブ効果 |
| :--- | :--- | :--- |
| **攻撃型** | 攻撃 / 会心 | 固定ダメージ、ダメージ%上昇 |
| **防御型** | HP / 防御 | 被ダメージ軽減、シールド強度 |
| **サポート型** | リジェネ / 速度 | クールダウン短縮、回復量% |

**ルーンオーブシステム：**
高品質なルーンを装備することで、「オーブ」（青、赤、または緑）が蓄積されます。特定のオーブのしきい値（例：20オーブ）に達すると、追加の攻撃力+2%や特定の属性ダメージブーストなどの**高度な効果**が解放されます。レジェンダリー（伝説）ルーンは最も多くのオーブを提供し、ティア2およびティア3の効果を有効にするために必要です。

<Callout type="warning" title="資源管理">
携帯アイテムをレベル15までアップグレードするには、大量の資源が必要です。素材をチーム全体に分散させる前に、まずは主力となる戦闘アニモに集中して使用しましょう。
</Callout>

</Section>
