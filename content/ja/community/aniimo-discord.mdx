<Section icon={Sparkles} title="Aniimoの変種と絆システム">

**Aniimo Discord**での大きな議論のポイントは、絆システムの進化です。プレイヤーはAniimoと一緒に戦う以上のことを望んでいます。撫でたり、餌をあげたり、一緒に時間を過ごしたりすることで、プレイヤーは**絆レベルシステム**を解放できます。これにより、信頼と訓練を通じて、地上型のAniimoが登り方を学ぶなどのユニークなスキルにアクセスできるようになります。

さらに、**変種**の需要が急増しています。標準的なサンダーストームバージョンを超えて、プレイヤーは単なるステータス強化ではなく、真のトロフィーのように感じられる季節限定の衣装や虹色のPrismanaフォームを求めています。

| 変種タイプ | 説明 | 入手方法 |
| :--- | :--- | :--- |
| **Prismana** | 虹色の天体のような外見 | 稀な孵化または高ティアの進化 |
| **Thunderstorm** | 電気で満たされた属性フォーム | アクティブな嵐のイベント中に出現 |
| **Seasonal** | 休日テーマ（例：ジャック・オー・ランタン） | 期間限定の年次イベント |
| **Darkling** | 影に満ちた神秘的なフォーム | ダークゾーンの探索または特別なクエスト |

<Section icon={Settings} title="マルチプレイヤーとQoL（品質向上）の修正">

マルチプレイヤーでの交流は、**Aniimo Discord**コミュニティの要です。共通の不満は、レベル差のために友達と一緒に遊べないことです。提案されている**レベルマッチング**システムは、プレイヤーをゾーンやパーティリーダーのレベルに自動的にスケールさせ、誰にとってもコンテンツを適切なものに保ちます。さらに、「エッグハイスト（卵強奪）」モードも精査されており、このモードが自分勝手なソロの稼ぎ場にならないよう、公平な戦利品分配を求める声が上がっています。

| 機能エリア | リクエストされた変更 | メリット |
| :--- | :--- | :--- |
| **協力プレイ** | 自動レベル同期 | 進行状況に関係なく友達とプレイ可能 |
| **エッグハイスト** | 共有戦利品分配 | 溜め込みよりもチームワークを促進 |
| **インベントリ** | Antipodの制限撤廃 | 長時間の狩猟セッション中の摩擦を軽減 |
| **UI/UX** | ローカルタイムゾーン表示 | 時間指定の天候イベントを追跡しやすくする |

**その他のQoL改善:**
- **リモートショッピング**: 外出先でアイテムを購入できるよう、BeaniebotをホームランドやRVに追加。
- **ダイアログ制御**: 読書スピードに合わせてプレイヤーがテキスト送りを制御可能。
- **シャイニー通知**: 光るAniimoが近くにいる際、より大きく視覚的な合図を出す。
- **カスタマイズ**: 服や家具の完全なカラーコントロール（木目調や光沢エフェクトを含む）。
