<Section icon={Zap} title="설치 및 Xbox Play Anywhere 설정">

**Xbox용 애니모**를 시작하는 방법은 간단하지만, Xbox 생태계의 이점을 극대화하려면 몇 가지 특정 단계가 필요합니다. Xbox 플레이어에게 가장 큰 장점 중 하나는 **Xbox Play Anywhere** 통합입니다. 즉, 콘솔에서 게임을 다운로드하면 Windows PC용 디지털 버전도 소유하게 되며, 모든 도전 과제, 저장 데이터 및 DLC가 자동으로 동기화됩니다.

| 요구 사항 | 최소 사양 | 권장 사양 |
| :--- | :--- | :--- |
| **저장 공간** | 45 GB | 60 GB (향후 업데이트 대비) |
| **인터넷** | 상시 연결 | 고속 광대역 |
| **구독** | Game Pass (멀티플레이어용) | Game Pass Ultimate |
| **계정** | Microsoft 계정 | 2단계 인증이 설정된 Microsoft 계정 |

아이딜 대륙은 반응형 세계입니다. 특정 애니모는 특정 기상 현상이나 시간대에만 나타납니다. 예를 들어, 희귀한 전기 타입 애니모는 고산 지대의 뇌우 중에만 나타날 수 있습니다. **애니포드(Aniipods)**를 사용하고 함정을 설치하려면 정확한 타이밍이 필요하며, 이는 Xbox 무선 컨트롤러의 햅틱 피드백을 통해 더욱 강화됩니다.

| 메카닉 | 설명 | 컨트롤러 단축키 |
| :--- | :--- | :--- |
| **트와인(Twine)** | 슬롯에 있는 애니모로 변신 | 오른쪽 범퍼 (RB) |
| **포획(Capture)** | 약해진 적에게 애니포드 투척 | 왼쪽 트리거 (LT) + X |
| **RV 소환** | 이동식 기지 접속 | 뷰 버튼 (Select) |
| **스킬 1/2** | 애니모 전용 능력 사용 | A / B 버튼 |

<Callout type="success" title="전투 전략">

<Section icon={Map} title="홈랜드 및 소셜 시스템">

전투 외에도 **Xbox용 애니모**는 "로드트립" 라이프스타일을 강조합니다. 모든 패스파인더는 이동식 주택 역할을 하는 개인용 RV를 소유합니다. 이 시스템을 통해 오픈 월드 내에서 상당한 창의적 표현과 소셜 상호작용이 가능합니다. 아이딜 전역에 지정된 캠핑장에 RV를 주차하여 다른 플레이어를 만나고, 자원을 거래하거나, 단순히 자신의 컬렉션을 뽐낼 수 있습니다.

**홈랜드 활동:**
- **농사**: 애니모에게 버프를 제공하는 희귀 작물을 심습니다.
- **꾸미기**: 공중 대도시에서 찾은 가구를 사용하여 RV 내부를 커스터마이징합니다.
- **소셜 활동**: 친구들을 홈랜드로 초대하여 미니 게임을 즐기고 협동 계획을 세웁니다.
- **애니모 상호작용**: 포획한 크리처들을 캠프에 풀어놓아 행복도를 높입니다.

| 건물 유형 | 주요 기능 | 자원 비용 |
| :--- | :--- | :--- |
| **정원 구역** | 소모성 버프 아이템 재배 | 낮음 (나무/씨앗) |
| **애니모 우리** | 펫의 패시브 XP 획득 | 중간 (돌/철) |
| **제작대** | 고급 애니포드 제작 | 높음 (희귀 광석) |

<Callout type="info" title="소셜 기능">
Xbox 버전은 "콘솔 키보드 및 마우스" 입력을 지원하며, 이는 공중 대도시에서 다른 패스파인더와 채팅하거나 정교한 홈랜드 장식에 이름을 붙일 때 특히 유용합니다.
</Callout>
