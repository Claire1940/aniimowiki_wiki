<Section icon={Shield} title="ブレイクゲージとひるみ（スタッガー）メカニクスのマスター">

**アニモ PvP**で最も重要なメカニクスは**ブレイクゲージ**です。このゲージは盾の役割を果たし、アクティブである限り、あなたのアニモは激しい猛攻に耐えることができます。しかし、このゲージが枯渇するとキャラクターは「ひるみ（スタッガー）」状態になり、「大ダメージ」コンボやアルティメットに対して無防備になります。容赦のないプレイヤーは、このガード崩しを即座に利用し、完璧なタイミングの一連のコンボで一気に勝負を決めてきます。

相手のゲージを削ることと同様に、自分のゲージを管理することも極めて重要です。タイミングの良い回避やブロックなどの防御的なプレイでゲージを維持し、攻撃的な「ポーク（牽制）」で敵の防御を徐々に削りましょう。もし自分のゲージが低下していることに気づいたら、一旦交代して控えで安全に回復させるのが最善の策となることが多いです。

| リソース | 主な機能 | 回復方法 |
| :--- | :--- | :--- |
| **HP（体力）** | 生存能力の総量 | ヒーラー（グレイシー）またはパッシブアイテム |
| **ブレイクゲージ** | 体勢維持とスタッガー防止 | 被弾していない間に自動回復 |
| **エネルギー／スキル** | 特殊アビリティの発動 | 攻撃のヒット、または時間経過による回復 |
| **アルティメットチャージ** | 試合を決定づける強力な技 | ダメージを与える、または被弾する |

<Section icon={Sparkles} title="メタチーム編成：ソラリス、グレン・サラ、グレイシー">

**アニモ PvP**で勝利するチームを構築するには、単に高火力なユニットを選ぶだけでは不十分です。回復、シールド、そしてバーストダメージをカバーできる編成が必要になります。現在、2026年ベータメタの「三種の神器」は、スタッガー状態の隙を効果的に突くことができるユニットを中心に構成されています。

**ソラリス（Solaris）**はトップクラスのアタッカーとして君臨しています。そのアルティメットは、相手が「ステップ」で回避できない場合、HPバーを一瞬で消し去るほどの威力を持ちます。**グレン・サラ（Glen Sarah）**（しばしば脅威と呼ばれます）は、相手を常に守勢に立たせるために必要なプレッシャーを提供します。最後に、**グレイシー（Glacy）**は最高峰のヒーラーであり、ヒーラーがいなければ、3分間の試合の中でチームは最終的に削りダメージに屈することになります。

| ティア | アニモユニット | 主な役割 | 主な強み |
| :--- | :--- | :--- | :--- |
| **S+** | ソラリス | バーストDPS | 絶大なアルティメットダメージと範囲攻撃（AOE） |
| **S** | グレン・サラ | プレッシャー／ブルーザー | 高いブレイクダメージと脅威的なコンボ |
| **A+** | グレイシー | 専属ヒーラー | 長期戦においてチームを維持・回復する |
| **A** | ニンビー | サポート／シールド | 交代クールダウン中の安全を確保するシールド |
| **B+** | ザク | タクティカル／ポーク | 削りダメージと相手の読みを妨害するのに適する |

<Callout type="tip" title="プロのシナジーのコツ">
ソラリスに交代する前に、ニンビー（Nimby）を使ってシールドを展開してみてください。これにより、ソラリスの高火力アビリティの初期動作中が保護され、自分のターンを最大限に活かすことができます。
</Callout>

</Section>

<Section icon={Target} title="高度な戦術：交代とアルティメットのタイミング">

ハイレベルな**アニモ PvP**において、上級者と初心者の違いはキャラクター交代の扱いにあります。交代には特定のクールダウンがあります。無計画にキャラクターを交代させてしまうと、HPの低いユニットのまま逃げ場を失い、立ち往生することになります。実力のあるプレイヤーは、交代システムを敵のアルティメットを「回避」するため、あるいは自分のブレイクゲージが破壊されそうな時に新しいゲージを持つアニモを出すために使用します。

アルティメットのタイミングは、高度なプレイにおける第二の柱です。ほとんどのアルティメットには長い「予備動作」があるため、お互いがニュートラルな状態では簡単に回避されてしまいます。最適な戦略は、**ガードブレイク**を待つことです。相手がひるんでいる間は移動できないため、最も強力な攻撃を叩き込む絶好のチャンスとなります。

<Steps>
  <Step num="1" title="ポーク（牽制）フェーズ">
    自分のゲージを監視しつつ、隙の少ない素早い攻撃を使って相手の**ブレイクゲージ**を削ります。
  </Step>
  <Step num="2" title="ガードブレイク">
    ゲージがほぼ空になったら、重いコンボを叩き込んでスタッガー（ひるみ）状態を引き起こします。
  </Step>
  <Step num="3" title="アルティメットフィニッシュ">
    相手がひるんでいる間に、即座にアルティメットを発動します。これにより100%の命中率が保証されます。
  </Step>
  <Step num="4" title="リセット">
    もし相手が生存した場合は、ヒーラーやシールド役に交代して回復を図りつつ、DPSユニットのクールダウン回復を待ちます。
  </Step>
</Steps>

<Callout type="info" title="戦術的なメモ">
試合時間は通常3分に設定されています。時間内に決着がつかない場合は、残りのHPの割合と与えた総ダメージ量によって勝者が決定されます。
</Callout>

</Section>

このチームワーク重視のデザインは戦闘にも引き継がれており、プレイヤー間でブレイクのタイミングを合わせることで、ボスを常にひるませ続けることができます。これらのマルチプレイヤー要素のクオリティは非常に洗練されており、一部のベテランプレイヤーは、この体験を『ポケットモンスター エメラルド』のようなクラシックな名作や、現代のヒット作『Temtem』と比較して評価しています。
